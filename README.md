# Food Recipe Zone

Animated static recipe website for exploring famous dishes from all 28 Indian states.

Open `project.html` or `index.html` to start.

- `project.css` styles the animated landing page and state cards.
- `punjabfood.css` is the shared animated state-page style.
- `recipe.css` is the shared animated recipe-page style.
- Each state has a food intro page and a matching recipe page.
